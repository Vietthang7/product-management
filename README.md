# Tên Dự Án
Đây là một dự án quản lý sản phẩm và có tích hợp Socket.IO để chat Realtime
## Installation
Để cài đặt dự án, hãy chạy các lệnh sau:  
npm install
## Usage
Để khởi chạy dự án, chạy:  
npm start
